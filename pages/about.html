About & Source

What is BB?

BB — Boris Best is a public, curated knowledge base of best practices for Claude Code. It aggregates tips, patterns, workflows, and insights from Boris Cherny (Head of Claude Code at Anthropic) and the broader Claude Code community.

The knowledge base is built on top of the open-source repository shanraisshan/claude-code-best-practice curated by Shan Raisshan, which aggregates tips shared by Boris and others publicly.

BB is a module of a larger internal system called CEM (Claude Environment Manager) — this site is its public-facing layer, published as a free educational tool for developers using Claude Code daily.

Attribution

BB aggregates content from others. Every tip, quote, and excerpt carries explicit attribution to its original author and source link. This is both an ethical and legal requirement.

Disclaimer

BB is an independent, community-built project. It is not affiliated with, endorsed by, or officially connected to Anthropic.

The name "Boris Best" is a working title referencing Boris Cherny, Head of Claude Code at Anthropic, whose publicly shared tips form the core of this knowledge base. Use of his name is descriptive and for attribution purposes only — it does not imply any official relationship or endorsement.

Claude Code evolves rapidly. Some tips may reference deprecated flags or workflows. Each tip links to its original source — always check the source for the most current information.
